# Программа для скачивания случайного комикса xkcd по программированию на пайтоне и публикации в телеграмм-канал

Программа скачивает случайные страницы комикса и отправляет их в телеграмм канал

## Что находится в репозитории:

Для работы программы в репозитории находится следующие файлы:

1. `main.py` - главный скрипт с кодом python. Тут написана логика работы и все запросы с обработками
2. .`env` - Файл, куда мы будем вносить переменные окружения для их хранения
3. `.gitignore` - Файл, где мы говорим python что игнорировать
4. `requirementx.tx`t - Файл где записаны все модули и библиотеки, необходимые для работы программы

## Зависимости:

Скрипт использует следующие модули и библиотеки для работы:

- `requests==2.24.2`
- `python-dotenv==1.2.2`
- `pyTelegramBotApi==4.35.2`

## Что понадобится для работы?

Программа написана на версии `python 3.13.7`. Для запуска выполните следующее:
- Скачайте репозиторий к себе на компьютер
- Откройте консоль и пропишите туда: `pip install -r requirementx.txt`. Команда установит все нужные зависимости для запуска программы
- Вам потребуется получить API-Токен телеграмм бота. Для этого в поиске телеграмм введите "BotFather" (Отец всех ботов телеграмма). Он выдаст вам API-Ключ.
- Дальше открываем файл .env и прописываем туда наш ключ таким образом: ```TELEGRAM_TOKEN=Ваш_Токен```
- Так же в поисковике ищем бота @TelegramBotRaw. Перешлите ему случайное сообщение из группы и он вышлет вам chanel_id вашего канала.
- Открываем .env и прописываем туда наш chanel_id: ```CHANEL_ID=Ваш_chanelid```
- Последнее, это добавить в файл .end наш путь до директории, куда скрипт устанавливает случайный комикс: ```PATH_TO_DIRECTORY="Ваш_Путь"```

## Запуск

После добавления Токена, id канала и путей скачивания можете смело запускать скрипт:

**Запускается скрипт командой `python main.py`**
